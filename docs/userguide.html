User Documentation for Bodgitt and Scarper, LLC Application

Introduction and Index

This document describes how to install, launch and use the application. This user guide is also available online with the user interface. The images present in this user guide are stored in the docs\guide_images folder located at the same level as this file.

This guide is divided into the following sections:

  • Application Installation - Explains how to install the application
  • Application Executation - Describes how to run the program
  • Standalone Mode - Configuration in Standalone mode
  • Server Mode - Explains how to configure and run the server application
  • Network Mode Describes how to configure the network mode and how to run the client application

Network Mode

Network Configuration

After the appropriate command for running the application in the network is executed, the following configuration window is shown.

This menu, allows the user to configure the hostname or ip address of the server to connect and the port number where the server is listening requests. The Change button in this case allows the user to choose a mapped network drive.

Application Main Window

After the user as terminated the configurations and pressed the Connect button, the main application window is made visible.

The table with the database information is empty to avoid problems if the database had an extremely large size.

Search for Services

To search for all the services in the database the user should press the Search button.

Filter the Results

The previous search returned all services in the database, moste of the times the user will want to search for more specific services, that can be accomplished using the filters. In the image bellow the search was made by filling the Name and Hourly Charge fields and pressing in the Search button.

Clear the Filters

To clear the previous chosen filters the user should press the Clear button and the table is also updated with all services in the database.

Booking

Besides searching for services the user will mainly book the same services. The booking can be done by selecting a desired line in the table and than pressing the Book button. A popop menu will be presented to the user where it will be possible to introduce the customer id that is booking the selected service.

If the booking is successfull, the Customer ID column will be updated in the table with the respective number.

Returning a Service

For returning a service the user should, again, select the desired service in the table and than press the Return button.
